Форсунки по культурам

Компактные инжекторные форсунки на сегодняшний день являются наиболее востребованным типом форсунок для химобработки. Оптимальное сочетание невысокой цены и высокой эффективности позволило использовать компактные инжекторные распылители для многих технологических операций по внесению СЗР. Эти форсунки чаще всего применяются для внесения системных гербицидов по зерновым и пропашным культурам, а также по овощным культурам на начальных фазах вегитации. Инжекторные форсунки имеют цветовую и цифровую кодировку.
