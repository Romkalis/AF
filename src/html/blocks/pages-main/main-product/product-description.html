Описание форсунки IDK

Форсунки IDK являются самыми распространенными инжекторными распылителями для опрыскивателей. Это обусловлено невысокой ценой и возможностью работы при ветре до 3 м/с. Принцип работы инжекторной форсунки позволяет формировать более крупную каплю, которая при ударе об лист разбивается на большое количество более мелких капель. ... Полное описание

Наилучшие результаты распылитель IDK дает при следующих обработках:

Гербициды: предпосевные, довсходовые, послевсходовые системные.
Фунгициды: системные.
Инсектициды: системные.

Могут также применяться для жидких удобрений, регуляторов роста и полива. Используются на навесных, прицепных и самоходных опрыскивателях.

Форсунки IDK различаются по размеру и имеют соответствующую цветовую маркировку. Для подбора размера форсунки воспользуйтесь таблицей зависимостей скорости, давления и расхода.

Артикулы других производителей: ZF669, ZF668.
